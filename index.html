Memory Game Guide

How good is your memory?

Red, Blue, Green squares appear randomly on a 3 x 3 grid one after the other when a game has started. They are visible for around one second and then disappears.

You need to correctly enter in how many red, blue, and green squares appeared. The game starts with three squares that appear randomly on the board.

For each level, there are three input fields:

  • Input field one: Enter in how many red squares appeared on the game board.
  • Input field two: Enter in how many green squares appeared on the game board.
  • Input field three: Enter in how many blue squares appeared on the game board.

If you get the answer wrong, the game ends.

If you get the answer right, you can progress to the next stage.

Good luck!
