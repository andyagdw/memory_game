Memory Game

Memory Game Guide

How good is your memory?

Red, Blue, Green squares appear randomly on a 4 x 4 grid one after the other when a game has started. They are visible for one second and then disappears.

Each level has three input fields for entering the number of squares that appeared for each colour.

For each incorrect answer, you lose one life. When you reach zero lives, the game ends.

If you get the answer right, you can progress to the next stage. The game gets harder by increasing the number of squares that appear on the board by one for every level.

You need to correctly enter in how many red, blue, and green squares appeared. The game starts with three squares that appear randomly on the board.

Good luck!
